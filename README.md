# Student Management System

This project is a basic student management system designed to practice
systems thinking, data handling, and structured programming.

## Project Purpose
The goal of this project is to build a simple but functional system
that allows managing student information in an organized way.

## Features (Planned)
- Add new students
- View student records
- Update student information
- Delete student records
- Store data using a database

## Technologies
- Python
- SQL
- Basic system architecture

## Learning Objectives
- Understand how systems are structured
- Practice data modeling
- Improve logical thinking and clean code practices

## Project Status
In progress
